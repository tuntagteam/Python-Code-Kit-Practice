A cleaning robot moves around a smart house.

* Negative number = lose battery

## Input

Second line: n integers

## Output

## Example

### Input

# hint
## 11. Robot Longest Safe Road — Hint

* Use a variable to count the current consecutive positive numbers
* If the current number is greater than 0:

  * increase the current count
* Otherwise:

  * reset the count to 0
* Keep another variable for the maximum length found so far
* Compare every time after increasing the count

Think about:

```python
current_length
max_length
```

---

## 12. Magic Score Combo — Hint

* Try every pair of numbers
* Use 2 loops
* Check if:

```python
arr[i] + arr[j] == K
```

* If you find one pair:

  * print `"YES"`
  * stop the program
* If no pair works after checking everything:

  * print `"NO"`

Important:

* `i` and `j` must not be the same position
* Start the second loop from `i + 1` to avoid duplicates
