# 1. ระบบคำนวณพลังงานรถไฟแม่เหล็ก

รถไฟแม่เหล็กความเร็วสูงต้องใช้พลังงานในการวิ่งผ่านแต่ละสถานี

หากสถานีใดมีค่าพลังงานเป็นลบ หมายถึงรถไฟเสียพลังงาน  
หากเป็นบวก หมายถึงได้รับพลังงานเพิ่ม

วิศวกรต้องการรู้ว่า

“ช่วงไหนของการเดินทางที่ทำให้รถไฟมีพลังงานรวมมากที่สุด”
Maximum Subarray Sum || Kadane's Algorithm

## Input
บรรทัดแรก จำนวนสถานี n

บรรทัดที่สอง ค่าพลังงานของแต่ละสถานี

## Output
ผลรวมพลังงานสูงสุดที่เกิดขึ้นต่อเนื่องกัน

## Example

### Input
7

4 -2 5 -1 2 -7 6

4 + -2 + 5 + -1 + 2 = 8

### Output
8

# 2. ระบบตรวจจับสายลับ

เมืองอัจฉริยะมีประชากรเรียงต่อกันเป็นแถว

แต่ละคนจะมีหมายเลขประจำตัว

เจ้าหน้าที่สงสัยว่า “สายลับ” จะเป็นคนที่มีหมายเลขปรากฏเพียงครั้งเดียว  
ในขณะที่คนทั่วไปจะมีหมายเลขซ้ำกัน 2 ครั้งเสมอ

จงหาหมายเลขของสายลับ

## Input
จำนวนเต็ม n

หมายเลขประชาชน n ตัว

## Output
หมายเลขของสายลับ

## Example

### Input
7

4 1 2 1 2 4 9

### Output
9

# 3. ระบบปล่อยจรวดอัตโนมัติ

ศูนย์อวกาศกำลังนับถอยหลังปล่อยจรวด

หากตัวเลขในลิสต์ “ลดลง” จากค่าก่อนหน้า  
แสดงว่าระบบผิดปกติทันที

จงตรวจสอบว่าค่าทั้งหมดเรียงจากน้อยไปมากหรือไม่

## Input
จำนวน n

ตัวเลข n ตัว

## Output
SORTED หรือ UNSORTED

## Example

### Input
5

1 2 2 5 9

### Output
SORTED

# 4. ระบบเก็บเหรียญของนักผจญภัย

นักผจญภัยเดินผ่านดันเจียนที่มีเหรียญหลายช่อง

แต่ละช่องมีจำนวนเหรียญแตกต่างกัน

นักผจญภัยต้องการเลือก “ช่องที่มีเหรียญมากที่สุด”

## Input
จำนวนช่อง n

จำนวนเหรียญในแต่ละช่อง

## Output
จำนวนเหรียญสูงสุด

## Example

### Input
6

4 9 2 11 7 1

### Output
11

# 5. ระบบคัดกรองผู้เล่นเกม

การแข่งขันเกมออนไลน์ต้องการคัดผู้เล่นที่มีคะแนนเกินเกณฑ์

หากคะแนนมากกว่าหรือเท่ากับ K ให้ผ่านเข้าสู่รอบต่อไป

จงหาว่ามีผู้เล่นผ่านทั้งหมดกี่คน

## Input
บรรทัดแรก จำนวนผู้เล่น n และค่า K

บรรทัดที่สอง คะแนนของผู้เล่นทั้งหมด

## Output
จำนวนผู้เล่นที่ผ่าน

## Example

### Input
5 80

75 90 81 60 100

### Output
3

# 6. ระบบประตูเลเซอร์

ห้องทดลองมีประตูเลเซอร์หลายบาน

ประตูแต่ละบานจะเปิดได้ก็ต่อเมื่อรหัสเป็น “เลขคู่”

จงแสดงจำนวนประตูที่สามารถเปิดได้

## Input
จำนวนรหัส n

รหัสทั้งหมด n ตัว

## Output
จำนวนเลขคู่

## Example

### Input
6

3 8 10 5 7 12

### Output
3

# 7. ระบบตรวจจับเสียงสะท้อน

นักวิทยาศาสตร์บันทึกคลื่นเสียงออกมาเป็นข้อความ

หากข้อความมีตัวอักษรซ้ำติดกัน  
แสดงว่าเกิด “เสียงสะท้อน”

## Output
YES หรือ NO

## Example

### Input
helloo

eye

YES

NO

# 8. ระบบแบ่งทีมแข่งขัน

ครูต้องการแบ่งนักเรียนออกเป็น 2 ทีม

หากจำนวนนักเรียนเป็นเลขคี่

จงแสดงจำนวนสมาชิกของทั้ง 2 ทีม

## Input
จำนวนนักเรียน n

## Output
จำนวนสมาชิกทีม A และทีม B

## Example

### Input
9

### Output
5 4

# 10. ระบบจัดเก็บกล่องสินค้าอัตโนมัติ

โกดังมีหุ่นยนต์เรียงกล่องเป็นชั้น

แต่ละกล่องมีน้ำหนักต่างกัน

หุ่นยนต์ต้องการรู้ว่า

“กล่องหนักที่สุดกับเบาที่สุด ต่างกันเท่าไร”

## Input
จำนวนกล่อง n

น้ำหนักกล่อง n ค่า

## Output
ผลต่างระหว่างค่าสูงสุดและต่ำสุด

## Example

### Input
5

10 25 7 18 30

### Output
23

## Input
จำนวนวัน n

แต้มสะสมแต่ละวัน

## Output
จำนวนวันที่แต้มเพิ่มขึ้น

## Example

### Input
6
10 12 9 15 15 20

3

หากครบทั้งสองเงื่อนไข ให้แสดง

STRONG

หากไม่ครบ ให้แสดง

WEAK

ข้อความรหัสผ่าน

## Output
STRONG หรือ WEAK

### Input
code123
