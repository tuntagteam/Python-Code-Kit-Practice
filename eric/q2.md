# แบบฝึกหัดโปรแกรมพื้นฐาน

เอกสารนี้รวบรวมโจทย์โปรแกรมภาษา Python เบื้องต้น จำนวน 10 ข้อ พร้อมตัวอย่าง Input/Output ที่ชัดเจน

---

## 1. พิมพ์ชื่อหลายครั้ง
รับชื่อ 1 ชื่อ และจำนวนเต็ม N จากนั้นพิมพ์ชื่อนั้น N ครั้ง

**Input**

```txt
Gun
3
```

**Output**

```txt
Gun
Gun
Gun
```

---

## 2. นับเลข 1 ถึง N
รับเลข N แล้วพิมพ์เลขตั้งแต่ 1 ถึง N

**Input**

```txt
5
```

**Output**

```txt
1
2
3
4
5
```

---

## 3. นับถอยหลัง
รับเลข N แล้วพิมพ์ถอยหลังจนถึง 1

**Input**

```txt
5
```

**Output**

```txt
5
4
3
2
1
```

---

## 4. ตรวจสอบผ่านหรือไม่ผ่าน
รับคะแนน 1 จำนวน
- ถ้าคะแนนตั้งแต่ 50 ขึ้นไป ให้แสดง `PASS`
- ถ้าน้อยกว่า 50 ให้แสดง `FAIL`

**Input**

```txt
72
```

**Output**

```txt
PASS
```

---

## 5. เลขคู่หรือเลขคี่
รับเลข 1 จำนวน
- ถ้าเป็นเลขคู่ แสดง `EVEN`
- ถ้าเป็นเลขคี่ แสดง `ODD`

**Input**

```txt
9
```

**Output**

```txt
ODD
```

---

## 6. หาผลรวมเลข 1 ถึง N
รับเลข N แล้วหาผลรวมตั้งแต่ 1 ถึง N

**Input**

```txt
5
```

**Output**

```txt
15
```

---

## 7. สูตรคูณแม่เดียว
รับเลข 1 จำนวน แล้วแสดงสูตรคูณแม่ของเลขนั้นตั้งแต่ 1 ถึง 12

**Input**

```txt
2
```

**Output**

```txt
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
...
2 x 12 = 24
```

---

## 8. เดารหัสผ่าน
กำหนดรหัสผ่านคือ `1234` ให้ผู้ใช้กรอกไปเรื่อย ๆ จนกว่าจะถูก แล้วแสดง `Correct`

**ตัวอย่าง**

```txt
1111
2222
1234
```

**Output**

```txt
Correct
```

---

## 9. พิมพ์ดาวเป็นแถว
รับเลข N แล้วพิมพ์ `*` ตามจำนวน

**Input**

```txt
5
```

**Output**

```txt
*****
```

---

## 10. พิมพ์สามเหลี่ยมดาว
รับเลข N แล้วพิมพ์ดาวเพิ่มทีละแถว

**Input**

```txt
4
```

**Output**

```txt
*
**
***
****
```
