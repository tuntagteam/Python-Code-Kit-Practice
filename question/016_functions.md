# Functions

## Goal

Practice grouping steps into reusable Python functions.

## What students should know first

* Know variables.
* Know if statements.
* Know loops.

## Questions

### Question 1: Hello Helper

**Problem description:** Write a function named hello that prints Hello, camper! Call it once.

**Input format:** No input.

**Output format:** Greeting line.

**Example input:**

```text
```

**Example output:**

```text
Hello, camper!
```

**Hint:** Define the function before calling it.

**Difficulty:** Easy
