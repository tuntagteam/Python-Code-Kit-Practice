# Variables

## Goal

Practice saving information in named boxes called variables and using those values later.

## What students should know first

* Know basic data types.
* Know how to print a value.
* Know that variable names should be clear.

## Questions

### Question 1: Name Tag Maker

**Problem description:** Store a camper name in a variable and print Hello followed by the name.

**Input format:** One name.

**Output format:** A greeting.

**Example input:**

```text
Nina
```

**Example output:**

```text
Hello Nina
```

**Hint:** Save the input in a variable before printing.

**Difficulty:** Easy

### Question 2: Pet Counter

**Problem description:** Store the number of pets in a variable. Add 1 new pet and print the new total.

**Input format:** One whole number.

**Output format:** The new total.

**Example input:**

```text
3
```

**Example output:**

```text
4
```

**Hint:** Use a variable like pets.

**Difficulty:** Easy

### Question 3: Robot Battery

**Problem description:** A robot has a battery percent. Store it and subtract 10 after a game. Print the remaining percent.

**Input format:** One whole number.

**Output format:** Remaining battery.

**Example input:**

```text
80
```

**Example output:**

```text
70
```

**Hint:** Convert input to int before subtracting.

**Difficulty:** Easy

### Question 4: Favorite Color Card

**Problem description:** Store a name and favorite color. Print a sentence joining both variables.

**Input format:** Two lines: name and color.

**Output format:** A sentence.

**Example input:**

```text
Sam
blue
```

**Example output:**

```text
Sam likes blue
```

**Hint:** Use two variables.

**Difficulty:** Easy

### Question 5: Score Swap

**Problem description:** Two players typed their scores. Store both scores, swap the values, and print them in the new order.

**Input format:** Two integers.

**Output format:** Two lines after swapping.

**Example input:**

```text
10
25
```

**Example output:**

```text
25
10
```

**Hint:** Use a temporary variable or Python swap.

**Difficulty:** Medium

### Question 6: Magic Doubling Coin

**Problem description:** A magic coin count doubles. Store the count, double it, and print the result.

**Input format:** One integer.

**Output format:** Doubled value.

**Example input:**

```text
6
```

**Example output:**

```text
12
```

**Hint:** count = count * 2 is useful.

**Difficulty:** Medium

### Question 7: Train Seats

**Problem description:** A train has seats and passengers. Store both values and print how many empty seats remain.

**Input format:** Two integers: seats and passengers.

**Output format:** Empty seats.

**Example input:**

```text
40
27
```

**Example output:**

```text
13
```

**Hint:** Subtract passengers from seats.

**Difficulty:** Medium

### Question 8: Level Up Game

**Problem description:** Read a player's level and bonus. Add the bonus to the level, then print Level: followed by the new level.

**Input format:** Two integers.

**Output format:** Level message.

**Example input:**

```text
4
3
```

**Example output:**

```text
Level: 7
```

**Hint:** Update the level variable.

**Difficulty:** Medium

### Question 9: Three Box Sum

**Problem description:** Store three box numbers. Print their total and then print the biggest original box number.

**Input format:** Three integers.

**Output format:** Two lines: sum and biggest number.

**Example input:**

```text
5
8
2
```

**Example output:**

```text
15
8
```

**Hint:** Variables can hold each number before you compare them.

**Difficulty:** Hard

### Question 10: Mini Shop Receipt

**Problem description:** Store an item name, price, and quantity. Print the total cost sentence.

**Input format:** Three lines: item, price, quantity.

**Output format:** A total cost sentence.

**Example input:**

```text
pencil
4
6
```

**Example output:**

```text
6 pencil cost 24 coins
```

**Hint:** Multiply price by quantity.

**Difficulty:** Hard
