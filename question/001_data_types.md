# Data Types

## Goal

Practice recognizing Python values like text, whole numbers, decimal numbers, and True or False values.

## What students should know first

* Know how to open a Python file.
* Know that Python can store different kinds of values.
* Know how to run a simple program.

## Questions

### Question 1: Robot Value Sorter

**Problem description:** A classroom robot reads one value. Print what kind of value it is: string, integer, float, or boolean. Treat True and False as boolean.

**Input format:** One value.

**Output format:** The data type name.

**Example input:**

```text
42
```

**Example output:**

```text
integer
```

**Hint:** Check for True or False first, then try whole number and decimal number checks.

**Difficulty:** Easy

### Question 2: Space Badge Text

**Problem description:** A space camp badge has a name written on it. Print the name and say that it is text.

**Input format:** One name.

**Output format:** The name on one line and string on the next line.

**Example input:**

```text
Luna
```

**Example output:**

```text
Luna
string
```

**Hint:** Names are strings because they are text.

**Difficulty:** Easy

### Question 3: Snack Count Type

**Problem description:** A snack shop records how many cookies are left. Print the number and the word integer.

**Input format:** One whole number.

**Output format:** The number and its type.

**Example input:**

```text
18
```

**Example output:**

```text
18
integer
```

**Hint:** Whole numbers are integers.

**Difficulty:** Easy

### Question 4: Potion Decimal Type

**Problem description:** A magic potion bottle shows its liters using a decimal number. Print the amount and the word float.

**Input format:** One decimal number.

**Output format:** The decimal value and its type.

**Example input:**

```text
2.5
```

**Example output:**

```text
2.5
float
```

**Hint:** Numbers with a decimal point are floats.

**Difficulty:** Easy

### Question 5: Game Switch

**Problem description:** A game setting says whether music is on. Print boolean if the input is True or False.

**Input format:** One value: True or False.

**Output format:** boolean

**Example input:**

```text
False
```

**Example output:**

```text
boolean
```

**Hint:** Booleans have only two values: True and False.

**Difficulty:** Easy

### Question 6: Type Detective

**Problem description:** Read three values: a pet name, an age, and a height. Print the type name for each value.

**Input format:** Three lines: text, whole number, decimal number.

**Output format:** Three lines with string, integer, and float.

**Example input:**

```text
Milo
9
1.35
```

**Example output:**

```text
string
integer
float
```

**Hint:** Each line can be checked separately.

**Difficulty:** Medium

### Question 7: Toy Box Label

**Problem description:** A toy box stores a toy name and the number of toys. Print a sentence showing both values and their types.

**Input format:** Two lines: toy name and count.

**Output format:** A friendly sentence.

**Example input:**

```text
blocks
12
```

**Example output:**

```text
blocks is string
12 is integer
```

**Hint:** Use the input values inside printed text.

**Difficulty:** Medium

### Question 8: Boolean Door

**Problem description:** A treasure room opens only when the card says True. Read the card value and print boolean, then print open if it is True, otherwise closed.

**Input format:** One value: True or False.

**Output format:** Two lines: boolean and open or closed.

**Example input:**

```text
True
```

**Example output:**

```text
boolean
open
```

**Hint:** Compare the text to True.

**Difficulty:** Medium

### Question 9: Mixed Backpack

**Problem description:** Read four backpack items: a word, a whole number, a decimal number, and True or False. Print their type names separated by spaces.

**Input format:** Four lines.

**Output format:** One line of four type names.

**Example input:**

```text
map
3
4.5
True
```

**Example output:**

```text
string integer float boolean
```

**Hint:** Build one output line after finding each type.

**Difficulty:** Hard

### Question 10: Data Parade

**Problem description:** Read n values. For each value, print its best type name. Values may be True, False, whole numbers, decimals, or words.

**Input format:** First line n, then n values.

**Output format:** One type name per line.

**Example input:**

```text
5
7
rain
3.14
False
0
```

**Example output:**

```text
integer
string
float
boolean
integer
```

**Hint:** Use a loop and test each value in a careful order.

**Difficulty:** Hard
