# Python Practice Questions for Kids

This folder contains kid-friendly Python practice question files. The topics begin with beginner basics and slowly move toward algorithm practice.

## How to use these files

* Start from `001_data_types.md` and move in order.
* Each file has exactly 10 questions.
* Students should read the goal and prerequisite section before starting.
* Teachers can pick Easy questions for warm-ups, Medium questions for class practice, and Hard questions for challenges.
* The files include examples and hints, but not full solutions.
* Encourage students to test their programs with the example input first, then make their own extra tests.
