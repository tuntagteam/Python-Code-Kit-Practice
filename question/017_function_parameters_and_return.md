# Function Parameters and Return

## Goal

Practice sending values into functions and getting answers back.

## What students should know first

* Know basic functions.
* Know variables.
* Know expressions.

## Questions

### Question 1: Greeting Name

**Problem description:** Write a function that takes a name and returns Hello plus the name. Print the returned value.

**Input format:** One name.

**Output format:** Greeting.

**Example input:**

```text
Lia
```

**Example output:**

```text
Hello Lia
```

**Hint:** Return a string instead of printing inside the function.

**Difficulty:** Easy

### Question 2: Double Number

**Problem description:** Write a function that takes a number and returns double it.

**Input format:** One integer.

**Output format:** Doubled number.

**Example input:**

```text
6
```

**Example output:**

```text
12
```

**Hint:** Use return n * 2.

**Difficulty:** Easy

### Question 3: Add Two Gems

**Problem description:** Write a function that takes two numbers and returns their sum.

**Input format:** Two integers.

**Output format:** Sum.

**Example input:**


**Example output:**


**Hint:** Parameters hold the inputs.

**Difficulty:** Easy

### Question 8: Word Repeater

**Problem description:** Write a function that takes a word and count, then returns the word repeated count times with no spaces.

**Output format:** Repeated word.

**Example input:**

```text
ha
```

**Example output:**

```text

```

**Hint:** String multiplication can be returned.

**Difficulty:** Medium

### Question 10: Safe Divide

**Problem description:** Write a function that takes a and b. Return cannot divide if b is 0, otherwise return a / b.

**Input format:** Two integers.

**Output format:** Result or message.

**Example input:**

```text
8
0
```

**Example output:**

```text
cannot divide
```

**Hint:** Check b before dividing.

**Difficulty:** Hard
