# Print and Output

## Goal

Practice showing friendly messages, numbers, and results on the screen.

## What students should know first

* Know basic data types.
* Know what a string is.
* Know how to run Python code.

## Questions

### Question 1: Hello Camp

**Problem description:** Print exactly the welcome message for coding camp.

**Input format:** No input.

**Output format:** One welcome line.

**Example input:**

```text
```

**Example output:**

```text
Welcome to Python Camp!
```

**Hint:** Use print with the exact text.

**Difficulty:** Easy

### Question 2: Robot Face

**Problem description:** Print a three-line robot face.

**Input format:** No input.

**Output format:** Three lines of the robot face.

**Example input:**

```text
```

**Example output:**

```text
[o_o]
/| |\
 / \
```

**Hint:** Escape backslashes if needed.

**Difficulty:** Easy

### Question 3: Magic Words

**Problem description:** Print the words Spark, Shine, and Smile on separate lines.

**Input format:** No input.

**Output format:** Three lines.

**Example input:**

```text
```

**Example output:**

```text
Spark
Shine
Smile
```

**Hint:** Use three print calls or newline characters.

**Difficulty:** Easy

### Question 4: Score Board

**Problem description:** Read a name and score. Print them in scoreboard style.

**Input format:** Two lines: name and score.

**Output format:** Name: score.

**Example input:**

```text
Ari
15
```

**Example output:**

```text
Ari: 15 points
```

**Hint:** Print text and variables together.

**Difficulty:** Easy

### Question 5: Train Ticket

**Problem description:** Read a city and platform number. Print a ticket message.

**Input format:** Two lines: city and platform.

**Output format:** Ticket message.

**Example input:**

```text
Moonville
3
```

**Example output:**

```text
Train to Moonville leaves from platform 3
```

**Hint:** Keep spaces in the sentence.

**Difficulty:** Medium

### Question 6: Box of Stars

**Problem description:** Read a number n. Print one line containing n star characters.

**Input format:** One integer n.

**Output format:** A line of stars.

**Example input:**

```text
5
```

**Example output:**

```text
*****
```

**Hint:** String multiplication can repeat a character.

**Difficulty:** Medium

### Question 7: Pet List

**Problem description:** Read three pet names and print them as a numbered list.

**Input format:** Three lines.

**Output format:** Three numbered lines.

**Example input:**

```text
cat
dog
hamster
```

**Example output:**

```text
1. cat
2. dog
3. hamster
```

**Hint:** Include the numbers in the output.

**Difficulty:** Medium

### Question 8: Recipe Card

**Problem description:** Read a snack and an ingredient. Print a two-line recipe card.

**Input format:** Two lines.

**Output format:** A title line and ingredient line.

**Example input:**

```text
sandwich
cheese
```

**Example output:**

```text
Recipe: sandwich
Add: cheese
```

**Hint:** Match the punctuation exactly.

**Difficulty:** Medium

### Question 9: Mini Table

**Problem description:** Read two names and two scores. Print a small two-row table using a dash between name and score.

**Input format:** Four lines: name1, score1, name2, score2.

**Output format:** Two table rows.

**Example input:**

```text
Kim
8
Leo
11
```

**Example output:**

```text
Kim - 8
Leo - 11
```

**Hint:** Output can be plain text.

**Difficulty:** Hard

### Question 10: Banner Builder

**Problem description:** Read a word. Print it with a border of equals signs above and below. The border length should match the word length.

**Input format:** One word.

**Output format:** Three-line banner.

**Example input:**

```text
PYTHON
```

**Example output:**

```text
======
PYTHON
======
```

**Hint:** Use len(word) to choose how many equals signs to print.

**Difficulty:** Hard
