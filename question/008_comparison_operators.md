# Comparison Operators

## Goal

Practice comparing numbers and words with operators like greater than and equal to.

## What students should know first

* Know numbers and strings.
* Know if statements.
* Know operators like ==, !=, <, and >.

## Questions

### Question 1: Camp Warm Up

**Problem description:** Solve a small comparison operators task for coding camp.

**Input format:** Input values as described in the story.

**Output format:** Print the requested answer.

**Example input:**

```text
3
1 2 3
```

**Example output:**

```text
6
```

**Hint:** Break the story into small steps before coding.

**Difficulty:** Easy

### Question 2: Robot Helper

**Problem description:** Use comparison operators ideas to help a robot process a short list.

**Input format:** First line n, second line n integers.

**Output format:** Print one result.

**Example input:**


**Example output:**

```text

```

**Hint:** Loop through the list and update your answer.
