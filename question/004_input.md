# Input

## Goal

Practice reading what a user types and using it in a program.

## What students should know first

* Know variables.
* Know print output.
* Know that input from the keyboard starts as text.

## Questions

### Question 1: Friendly Echo

**Problem description:** Read one word and print it back with Hello in front.

**Input format:** One word.

**Output format:** Greeting line.

**Example input:**

```text
Maya
```

**Example output:**

```text
Hello Maya
```

**Hint:** input() reads text.

**Difficulty:** Easy

### Question 2: Age Next Year

**Problem description:** Read an age and print the age next year.

**Input format:** One integer.

**Output format:** Age plus 1.

**Example input:**

```text
10
```

**Example output:**

```text
11
```

**Hint:** Convert the input to int.

**Difficulty:** Easy

### Question 3: Snack Order

**Problem description:** Read a snack name and quantity. Print an order sentence.

**Input format:** Two lines: snack and quantity.

**Output format:** Order sentence.

**Example input:**

```text
muffin
4
```

**Example output:**

```text
You ordered 4 muffin
```

**Hint:** Quantity may stay as text if you only print it.

**Difficulty:** Easy

### Question 4: Two Number Add

**Problem description:** Read two numbers and print their sum.

**Input format:** Two integers.

**Output format:** The sum.

**Example input:**

```text
7
9
```

**Example output:**

```text
16
```

**Hint:** Both inputs must become integers.

**Difficulty:** Easy

### Question 5: Secret Code Word

**Problem description:** Read a secret word. If it is python, print Access granted. Otherwise print Try again.

**Input format:** One word.

**Output format:** Access result.

**Example input:**

```text
python
```

**Example output:**

```text
Access granted
```

**Hint:** Use an if statement after input.

**Difficulty:** Medium

### Question 6: Temperature Helper

**Problem description:** Read a temperature in Celsius and print warm if it is 25 or more, otherwise cool.

**Input format:** One integer.

**Output format:** warm or cool.

**Example input:**

```text
23
```

**Example output:**

```text
cool
```

**Hint:** Compare the number to 25.

**Difficulty:** Medium

### Question 7: Full Name Builder

**Problem description:** Read first name and last name. Print the full name on one line.

**Input format:** Two lines.

**Output format:** Full name.

**Example input:**

```text
Ada
Lovelace
```

**Example output:**

```text
Ada Lovelace
```

**Hint:** Add a space between the two parts.

**Difficulty:** Medium

### Question 8: Coin Total

**Problem description:** Read gold coins and silver coins. Gold coins are worth 10 points, silver coins are worth 1 point. Print total points.

**Input format:** Two integers.

**Output format:** Total points.

**Example input:**

```text
3
7
```

**Example output:**

```text
37
```

**Hint:** Multiply gold by 10.

**Difficulty:** Medium

### Question 9: Quiz Average

**Problem description:** Read three quiz scores and print the average as a decimal.

**Input format:** Three integers.

**Output format:** Average score.

**Example input:**

```text
8
9
10
```

**Example output:**

```text
9.0
```

**Hint:** Add the scores and divide by 3.

**Difficulty:** Hard

### Question 10: Many Names

**Problem description:** Read n, then read n names. Print Thank you, name for each one.

**Input format:** First line n, then n names.

**Output format:** One thank-you line per name.

**Example input:**

```text
3
Ana
Ben
Chai
```

**Example output:**

```text
Thank you, Ana
Thank you, Ben
Thank you, Chai
```

**Hint:** Use a loop after reading n.

**Difficulty:** Hard
