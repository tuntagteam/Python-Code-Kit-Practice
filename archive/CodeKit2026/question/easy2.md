# ชุดโจทย์ฝึกเขียนโปรแกรม

## 1. พลังงานสูงสุดต่อเนื่อง

นักวิทย์มีเซนเซอร์วัดพลังงานติดต่อกันหลายวัน จะหาระยะเวลาที่พลังงานรวมสูงสุดจากช่วงข้อมูลที่อยู่ติดกัน

- ข้อมูลอาจเป็นลบหรือบวก
- ต้องเลือกช่วงความยาวอย่างน้อย 1 ตัว

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (1 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นเลขจำนวนเต็ม N ตัว คั่นด้วยช่องว่าง

### Output

- ผลรวมสูงสุดของช่วงตัวเลขที่อยู่ติดกัน

### Example

```text
8
-3 4 -1 2 1 -5 4 6
```

```text
11
```

---

## 2. เลขที่หายไปจากลำดับ

มีเลข `1` ถึง `N` อยู่ในลำดับ แต่ข้อมูลขาดไป 1 ตัว ให้นักเขียนโปรแกรมหาว่าเลขไหนหาย

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (2 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นตัวเลข `N-1` ตัวจากลำดับ 1..N ที่ไม่เรียงและไม่ซ้ำกัน

### Output

- เลขที่หายไป

### Example

```text
5
1 2 4 5
```

```text
3
```

---

## 3. คะแนนตัวแรกที่ไม่ซ้ำ

มีคะแนนนักเรียนหลายคน เขียนลำดับคะแนนตามลำดับที่ประกาศ
ให้หาเลขคะแนนตัวแรกที่ปรากฏเพียงครั้งเดียวเท่านั้น

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (1 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นคะแนน `N` ตัว

### Output

- ค่าแรกของลำดับที่ปรากฏแค่ครั้งเดียว
- ถ้าไม่มี ให้พิมพ์ `-1`

### Example

```text
7
4 5 4 6 5 8 6
```

```text
8
```

---

## 4. หาคู่รวมเท่ากับ K

มีชุดตัวเลขให้มา 1 ชุด ต้องหาคู่ของ 2 ตัวที่รวมกันได้ `K`

### Input

- บรรทัดแรก มีจำนวนเต็มสองตัว `N` และ `K` (2 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นตัวเลข `N` ตัว

### Output

- ถ้ามีคู่ที่รวมได้ `K` ให้พิมพ์ `YES`
- ถ้าไม่มี ให้พิมพ์ `NO`

### Example

```text
6 10
2 4 7 3 8 1
```

```text
YES
```

---

## 5. ชนะติดกันยาวที่สุด

ในแต่ละวันมีคะแนนการแข่งขัน ถ้าคะแนนเป็นบวกถือว่า "ชนะ" หากคะแนนเป็น `0` หรือเป็นลบถือว่า "ไม่ชนะ"
หาจำนวนวันที่ชนะติดกันมากที่สุด

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (1 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นเลขคะแนน N ตัว

### Output
n
- จำนวนวันชนะติดต่อกันสูงสุด

### Example

```text
9
1 2 -1 3 4 5 0 6 7
```

```text
3
```

---

## 6. ลำดับเพิ่มขึ้นเข้มงวด

ตรวจสอบว่าลำดับตัวเลขเรียงจากน้อยไปมากอย่างเข้มงวดหรือไม่

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (1 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นเลข N ตัว

### Output

- `YES` ถ้าเป็นลำดับเพิ่มขึ้นเข้มงวด
- `NO` ถ้าไม่ใช่

### Example

```text
5
1 3 4 7 9
```

```text
YES
```

```text
5
1 3 3 7 9
```

```text
NO
```

---

## 7. นับคู่เลขเหมือนกัน

ให้เลขชุดหนึ่ง แล้วนับจำนวนคู่ `(i, j)` ที่ `i < j` และเลขที่ตำแหน่งทั้งสองเท่ากัน

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (1 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นเลข N ตัว

### Output

- จำนวนคู่ที่ตรงตามเงื่อนไข

### Example

```text
6
1 2 1 3 2 1
```

```text
4
```

---

## 8. รหัสลับสมดุล

ข้อความประกอบด้วยตัวอักษร `A` และ `B` เท่านั้น
ตรวจสอบว่าจำนวน `A` เท่ากับจำนวน `B` หรือไม่

### Input

- บรรทัดเดียว เป็นสตริงที่มีความยาวไม่เกิน 100000 ตัวอักษร

### Output

- `YES` ถ้าจำนวน `A` เท่ากับจำนวน `B`
- `NO` ถ้าไม่เท่ากัน

### Example

```text
AABBAB
```

```text
YES
```

---

## 9. ผลรวมสูงสุดจากหน้าต่าง K ตัว

ให้หา "ผลรวมมากที่สุด" ของตัวเลข `K` ตัวที่อยู่ติดกันในลำดับ

### Input

- บรรทัดแรก มีสองจำนวนเต็ม `N` และ `K` (1 ≤ K ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นตัวเลข N ตัว

### Output

- ผลรวมสูงสุดของหน้าต่างขนาด `K`

### Example

```text
7 3
2 5 -1 4 6 -2 3
```

```text
9
```

---

## 10. เกมเก็บเหรียญ

เดินจากช่องซ้ายสุดไปขวาสุดในแถวหนึ่ง หยิบเหรียญจากแต่ละช่องได้ แต่ห้ามหยิบจากช่องที่ติดกันสองช่อง
หาจำนวนเหรียญสูงสุดที่เก็บได้

### Input

- บรรทัดแรก เป็นจำนวนเต็ม `N` (1 ≤ N ≤ 100000)
- บรรทัดที่สอง เป็นจำนวนเต็ม N ตัวแทนจำนวนเหรียญในแต่ละช่อง

### Output

- จำนวนเหรียญสูงสุดที่สามารถเก็บได้

### Example

```text
6
5 1 2 10 6 2
```

```text
17
```
