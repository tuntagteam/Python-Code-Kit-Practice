# If Else

## Goal

Practice helping a program make simple choices.

## What students should know first

* Know variables.
* Know input.
* Know basic comparison ideas like bigger and smaller.

## Questions

### Question 1: Umbrella Check

**Problem description:** Read weather. If it is rain, print Take umbrella, otherwise print Enjoy outside.

**Input format:** One word.

**Output format:** Advice.

**Example input:**

```text
rain
```

**Example output:**

```text
Take umbrella
```

**Hint:** Compare the string.

**Difficulty:** Easy

### Question 2: Game Pass

**Problem description:** Read score. Print pass if score is at least 10, otherwise try again.

**Input format:** One integer.

**Output format:** pass or try again.

**Example input:**

```text
12
```

**Example output:**

```text
pass
```

**Hint:** Use >=.

**Difficulty:** Easy

### Question 3: Snack Enough

**Problem description:** Read cookies and kids. Print enough if cookies >= kids, otherwise need more.

**Input format:** Two integers.

**Output format:** enough or need more.

**Example input:**

```text
5
7
```

**Example output:**

```text
need more
```

**Hint:** Compare the two numbers.

**Difficulty:** Easy
